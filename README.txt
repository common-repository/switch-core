== Description ==
Switch Core is free WordPress plugin for developing different sections like sliders, about us, feature, team, testimonial, portfolio, contact, etc. It is a very light weight plugin that comes with clean codes. You can very easily create any professional section for your website and there is absolutely no hidden charges.

If you are in short of time and have to complete task for your project, you can try Switch core. We assure you, it wont let you down. Just install it and go to customizer option of WordPress and make any section you want.

== Installation ==

1. Upload `switch-core` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== FAQ ==


= Is Switch Core compatible with any WordPress themes? =
Yes of course! Switch Core will work with any WordPress themes, just select a page template like: "switch core page" from page then go to the theme main customizer. 

= Is Switch Core fully free? =
Yes! Switch Core is absolutely free and provided to you by Themesgrove. There is no hidden charge, if you wish you can give small donation.

= As a theme developer, how can Switch Core help me? =
Switch Core turns your theme customizer to get options to reduce development time. 

= What is the requirements to use Switch Core? =
Minimum Requirements
WordPress version 4.0 or greater.
PHP version 5.4 or greater.
